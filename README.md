# EmoLabMaker-ae

<img width="1280" height="720" alt="image" src="https://github.com/user-attachments/assets/b9d60e27-30a8-4e8a-91da-5b4fccd808cb" />

`EmoLabMaker` は、After Effects 上で

- 表情切り替え
- lab ファイルを使った口パク
- PSDToolKit 立ち絵 PSD の自動セットアップ（目パチ含む）
- 立ち絵の階層をまとめて切り替える「立ち絵」ビュー

ができます

もともとの `emo2layer` と `lab2layer` を統合した版で、現在は「立ち絵 / 口パク / 目パチ / PSD / 詳細」の
5 つのタブを切り替えて使えます（日常操作は「立ち絵」タブが中心。「詳細」は PSDToolKit を使わない手動向けのレガシーモード）

> [!WARNING]
> 現在 β 版です。とりあえず動くと思うけど、実験的範疇です  
> ある程度バグを取り除いてますが、私が使えればOKみたいな作りしているので、使用感が絶望的かも……？  

> [!IMPORTANT]
> x.y.z の x のバージョンが更新されたら破壊的変更（それまでのデータとは互換性がない）です。yは機能追加、zはバグ修正

## インストール方法

1. `EmoLabMaker.jsx` をダウンロード
2. 以下のいずれかの場所に配置:
   - **Windows**: `C:\Program Files\Adobe\Adobe After Effects <version>\Support Files\Scripts\ScriptUI Panels\`
   - **macOS**: `/Applications/Adobe After Effects <version>/Scripts/ScriptUI Panels/`
3. After Effects を再起動
4. メニューから `ウィンドウ` → `EmoLabMaker.jsx` を選択

> [!NOTE]
> mac 環境での動作チェックはしていないので動かなかったら改変してください


# emo2layer-ae (レイヤー選択タブ)

<img width="1655" height="887" alt="image" src="https://github.com/user-attachments/assets/6f4201d0-63bd-4c69-a8dc-47eba608f6e5" />

鳴花ミコト「なんか動いた」　　鳴花ヒメ「こわ～」

After Effects 用の表情レイヤーの表示切り替えツール。表情以外もOK  
指定したレイヤーを登録して、任意の場所で、不透明度エクスプレッションを一括設定します

試した中で一番高速だったマーカー制御を使用しています。今のところあまり困っていないので選択は排他的です（1つ以外すべて非表示）  
そのうちラジオボタンとかに対応できたら面白いわね

## 概要

emo2layer は、大量のレイヤーを分割したりエクスプレッションを使ったりして、  
表情アニメーションをかんたんに行えるスクリプトです

キーフレームで表示非表示しなくてもよくなります

> [!NOTE]
> 表情パーツがレイヤーわけされている必要があります  

> [!NOTE]
> PSDToolKit 対応など、ニコニコ動画などにおいてある立ち絵はだいたい多分OKです

## 機能

- **制御レイヤーの生成**  
  マーカー付きのヌルレイヤーを自動作成（1レイヤーでOK）
- **不透明度エクスプレッション**  
  選択したレイヤーに表情制御レイヤー連動の不透明度エクスプレッションを自動設定
- **自動判定**  
  設定済みデータの自動検知、登録済みレイヤーのみの表示
- **表情切り替え**  
  ボタン一つで表情変更

#### 基本の流れ

1. `対象` に、切り替えたいレイヤーが入っているコンポを選ぶ
2. `制御` に、マーカーを書き込むコンポを選ぶ（同じでも可）
3. `制御レイヤー作成` を押してレイヤーを作る（ `[Emo] <対象コンポ名>` が作られる）
4. 対象コンポをアクティブにして、切り替え対象レイヤーを選択して `登録`
5. パネルに並んだレイヤー名ボタンを押して、現在位置にマーカーを書き込む

#### 動作の仕組み

- 登録したレイヤーには不透明度エクスプレッションが設定されます
- 制御レイヤー上の最新マーカー名と同じ名前のレイヤーだけが表示されます
- マーカーは対象コンポの現在時刻に応じて参照されます
- `解除` を押すと、選択レイヤーのエクスプレッションを外して不透明度を `100` に戻します

#### 使いどころ

`通常` `笑い` `困り` `怒り` のように、レイヤー名そのものを状態名として扱う用途に向いています

#### 表情セット（一括切替）

目・口・眉など複数グループの表情をまとめて保存し、ワンクリックで切り替えられます
（PSDTool の「お気に入り」に近い使い方ができます）

1. 各グループの表情をマーカーボタンで作る
2. 「表情セット」パネルの `保存` → セット名を付ける
3. 切り替えたい位置に再生ヘッドを移動して `適用`
   → 全グループのマーカーが **制御コンポの現在時刻** に一括で書き込まれます

- セットは制御コンポ内の `[EmoSet] <名前>` ガイドレイヤーに保存されます（プロジェクトと一緒に保存・共有できます）
- 同名で保存すると上書きされます。`削除` でセットを削除できます

#### 補足

- `↺` ボタンで対象 / 制御コンポ一覧を再取得します
- 対象コンポをアクティブにしていないと `登録` と `解除` はできません
- 制御レイヤーが未作成のときは、マーカーボタンは押せません


# lab2layer-ae (口パクタブ)

<img width="1600" height="720" alt="thumb" src="https://github.com/user-attachments/assets/011f1b9f-25db-4ad2-b0dc-7362bbf4f2a7" />

鳴花ミコト「なんか動いた」　　鳴花ヒメ「こわ～」

After Effects 用の音素リップシンクツール。音素タイミングファイルを解析して音素レイヤーを生成し、不透明度エクスプレッションを一括設定します

## 概要

lab2layer は、音声合成ソフトが出力する lab ファイルを読み込み、After Effects で口パク（リップシンク）アニメーションを簡単に作成するためのスクリプトです

> [!WARNING]
> lab ファイルの構造が独自的で特殊だと動かない可能性があります（NEUTRINO なら `full.lab` ではなく `timing.lab` を使うなどなど）

## 機能

- **lab ファイルの解析**  
  lab ファイルから音素とタイミング情報を自動抽出
- **音素レイヤーの生成**  
  マーカー付きのヌルレイヤーを自動作成
- **不透明度エクスプレッション**  
  選択したレイヤーに音素連動の不透明度エクスプレッションを自動設定
- **音素フィルタリング**  
  必要な音素のみを選択して使用可能
- **複数音素対応**  
  レイヤー名にカンマ区切りで複数の音素を指定可能

## 使用例

1. VOICEVOX で「こんにちは」を書き出して、.lab ファイルを作る  
   ※設定で [lab ファイルを一緒に書き出す](https://voicevox.hiroshiba.jp/how_to_use/#:~:text=lab%20%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%E3%82%92%E6%9B%B8%E3%81%8D%E5%87%BA%E3%81%97) 方法
2. 「Browse...」で lab ファイルを読み込む
3. 使用したい音素にチェック  
   （基本はデフォルトの「Common」で OK。多くても閉じ口がわかりやすい s、m、r、w あたりにとどめておくとよさげ）

   > [x] a(1) [x] i(2) [x] o(1)  
   > [x] N(1) [x] pau(2) [ ] k(1)  
   > [ ] n(1) [ ] ch(1) [ ] w(1)

4. 口パク画像のあるコンポジション内で、再生ヘッドをセリフの開始位置に移動し、「Create」で音素レイヤーを配置
5. 口パク画像のレイヤー名を、表示したい音素名に変更
   - 閉じ口.png → レイヤー名: sil,N,cl
   - あ.png → レイヤー名: a
   - い.png → レイヤー名: i
   - お.png → レイヤー名: o,u

6. 変更した口パク画像のレイヤーを全選択 → 「Setup Opacity」でリアルタイム不透明度を設定

> [!NOTE]
> タイミングは「オフセット(ms)」（全体シフト＝映像先行の法則、既定 0）と「終了に閉じ口」（発話後に口を閉じる）で調整でき、次回起動時まで保存されます。
> `<` `>` ボタンは選択レイヤーのマーカーを 1 フレームずつ動かします

## 詳細な使い方

### Step 1: lab ファイルの読み込み

1. スクリプトパネルを開く
2. 「Browse...」ボタンをクリック
3. lab ファイル（`.lab`）を選択
4. 解析された音素がチェックボックス付きで表示される

### Step 2: 音素の選択

| ボタン     | 説明                           |
| ---------- | ------------------------------ |
| **All**    | すべての音素を選択             |
| **None**   | すべての選択を解除             |
| **Common** | よく使われている音素を自動選択 |

### Step 3: 音素レイヤーの作成

1. 使用する音素にチェックを入れる
2. タイムラインでセリフの開始位置に再生ヘッドを移動
3. 「Create」ボタンをクリック
4. `[P] ` から始まる音素レイヤー (ヌルレイヤー) が作成され、各音素がマーカーとして配置される

**音声レイヤーへのアタッチ:**

- 音声レイヤーを選択した状態で「Create」を実行すると、音声レイヤーに直接マーカーが追加されます
- こちらは、音声を移動するとマーカーも一緒に移動するので、位置調整がかんたんです

複数セリフの場合、各セリフごとに上記の手順を繰り返すことで、複数の Phoneme レイヤーを配置できます

現在時刻に有効なレイヤーのうち、一番上にあるものが参照される仕様です

### Step 4: 口パク画像の準備

各音素に対応する画像レイヤーを用意し、**レイヤー名を音素名に設定**します

```
// 一例
- あ.png  → レイヤー名: a
- い.png  → レイヤー名: i
- う.png  → レイヤー名: u,w
- え.png  → レイヤー名: e
- お.png  → レイヤー名: o
- 閉じ.png  → レイヤー名: N,sil,pau,def
   ⋮
```

> [!NOTE]
> 現在位置の音素レイヤーに対応する画像がなければ何も表示されません (現在位置が k なのに k 画像がないなど)  
> 現在位置に音素レイヤーがないときは「def」を含むレイヤーが表示されます

#### 複数音素を同じ画像で表示する場合

レイヤー名にカンマ区切りで複数の音素を指定できます

```
- a,e.png → レイヤー名: a,e（「あ」と「え」で同じ口の形を使用）
```

### Step 5: 不透明度エクスプレッションの設定

1. 口パク画像レイヤーをすべて選択
2. 「Setup Opacity」ボタンをクリック
3. コンポジション選択ダイアログで、音素レイヤー（`[P]` レイヤー）を配置する場所を選択
4. 各レイヤーに不透明度エクスプレッションが自動設定される

> [!TIP]
>
> - 探索が複数になるとプレビューが重くなるため、1 つのコンポジションのみ指定できるようにしています
> - 音素レイヤーを別のコンポジションに移動する場合は、再度「Setup Opacity」してください
> - 指定したコンポジションの開始位置と立ち絵の開始位置をあわせてください

<img width="1200" height="484" alt="スクリーンショット 2026-01-10 195327" src="https://github.com/user-attachments/assets/f267d0d3-8afd-44a5-a993-3777abdd3189" />

**エクスプレッションの仕組み:**

- 指定されたコンポジション内の `[P]` レイヤーを探索
- マーカーから現在の音素を取得（上のレイヤーが優先）
- レイヤー名と音素が一致したら不透明度 100、それ以外は 0
- カンマ区切りで複数音素に対応（例: `a,e` のレイヤーは a か e のときに表示）

## 口形状マッピング（PSDToolKit 互換・リネーム不要）

レイヤーを 1 枚ずつリネームする代わりに、「口形状マッピング」パネルで
**あ / い / う / え / お / ん(閉)** の 6 口形に音素を割り当てられます

1. 口パク画像のあるコンポをアクティブにして、口形レイヤーを選択
2. 各行の「割当」で口形に割り当てる（または「自動割当」でレイヤー名から推定）
3. 「適用」→ ダイアログで `[Lab]` レイヤーのあるコンポを選択

> [!TIP]
> 「自動割当」は、選択レイヤー名に「あ/い/う/え/お/ん/閉」が含まれていれば自動で振り分けます。
> PSD 立ち絵の口レイヤーは大抵この命名なので、選択 → 自動割当 → 適用 だけで済みます

**音素のプリセット（PSDToolKit 互換）:**

| 口形 | 音素 |
| --- | --- |
| あ | a |
| い | i |
| う | u, w |
| え | e |
| お | o |
| ん(閉) | N, cl, Q, pau, sil, br **+ どの口形にも属さない音素（子音など）** |

- 子音は自動的に「ん(閉)」になるため、列挙する必要はありません
- 音素リストは各行のテキストボックスで自由に編集できます（「プリセット」で初期値に復元）

**表情切替との共存:**

- レイヤー選択タブで登録済みのレイヤーに「適用」すると、**しゃべっている間だけ口を上書きし、
  しゃべっていない間は表情切替に従う**合成エクスプレッションになります
  （表情の口がそのまま生きるので、無音時に閉じ口固定になりません）
- 未登録のレイヤーに適用した場合は、しゃべっていない間「ん(閉)」が表示されます
- 「解除」で表情切替のみの状態（未登録だった場合は不透明度 100）に戻ります

## 音素について

**このスクリプトは特定の音素リストを持っていません。**  
lab ファイルから動的に音素を読み取るため、どんな音素でも対応可能です（たぶん maybe おそらく）

- **母音**　a, i, u, e, o
- **特殊**　N (ん), cl (っ), pau (ポーズ), sil (無音) 、br (ブレス)
- **子音**　k, s, t, n, h, m, y, r, w, g, z, d, b, p など
- 日本語……？ あ, か, さ, た, な, ー

## lab ファイルってなに？

lab ファイルは以下のように 3 つに区分けされた形式のテキストファイルです

```
0 1000000 sil
1000000 2500000 k
2500000 5000000 o
5000000 7500000 N
7500000 10000000 n
10000000 12500000 i
12500000 15000000 ch
15000000 17500000 i
17500000 20000000 w
20000000 22500000 a
22500000 30000000 sil
```

- 1 列目 → 開始時間（100 ナノ秒単位）
- 2 列目 → 終了時間（100 ナノ秒単位）
- 3 列目 → 音素

## 対応ソフトウェア

上のような、単純な形式の音素ファイルなら対応しています

すなわち、以下ソフトウェアが出力する lab ファイルにも対応しているはずです

- VOICEVOX
- NEUTRINO
- CeVIO / VoiSona
- A.I.VOICE
- その他 lab 形式を出力するソフトウェア


# PSDセットアップ (PSDタブ)

PSDToolKit（AviUtl 用）向けに作られた立ち絵 PSD を読み込んで、表情切替を自動セットアップします

## 使い方

1. **AE 標準の読み込みで PSD を追加**:
   `ファイル` > `読み込み` > 形式は「**コンポジション - レイヤーサイズを維持**」推奨
   （「コンポジション」でも動作します。「フッテージ」は不可）
2. PSD タブで `ルート` に読み込んだ PSD のコンポを選択
3. `制御` にマーカーを書き込むコンポを選択（通常はルートと同じで OK）
4. `解析してセットアップ / 更新` を押す
5. 確認ダイアログでセットアップするグループ（目 / 口 / 眉 など）を選んで実行

セットアップ後は、レイヤー選択タブの表情ボタンがすぐ使える状態になります

> [!NOTE]
> スクリプトは PSD の読み込み自体を行いません（バグ防止のため AE 標準機能に任せています）

## PSDToolKit 命名規則の対応表

| 記法 | PSDToolKit での意味 | EmoLabMaker での扱い |
| --- | --- | --- |
| `*レイヤー名` | ラジオボタン（排他表示） | 表情切替に自動登録。**プレフィックスは保持**し、表示時だけ base 名に整形 |
| `!レイヤー名` | 強制表示 | 表示状態を保証（登録はしない）。プレフィックスは保持 |
| `レイヤー名`（無印） | 任意指定（独立 ON/OFF） | 任意指定として登録 |
| `レイヤー名:flipx` / `:flipy` / `:flipxy` | 反転バリエーション（通常レイヤーとペア） | **対応**。`通常名` と `通常名:flipx` のペアを登録し、立ち絵タブの**グローバル反転ボタン**（↔左右 / ↕上下 / 通常）で立ち絵全体を反転。ペアの相手（base）が無い孤立 flip はスキップ |

> **反転の挙動について**: PSDToolKit の反転は「立ち絵全体の反転状態」で、その状態のときに `通常名` の代わりにペアの `通常名:flipx` が使われる仕様です。EmoLabMaker のグローバル反転ボタンは、これを **①ルートコンポの最上位レイヤーを中心線でミラー（Scale を反転＋位置を中心線で反転）＋ ②手描きペアの flip 側への差し替え** の2段で再現します。①は静的な値の書き換えだけなので**描画負荷ゼロ**です。反転状態はルートコンポのコメントに記録され、2回押しても二重反転せず「通常」で確実に戻ります。
>
> 反転レイヤーは絵描き側が用意したペア前提です。最上位レイヤーにキーフレーム/式/親子付けがあると正しくミラーできない場合があります（静的トランスフォーム前提）。
>
> **レイヤー名のカンマ「,」は使えません**: マーカーは「表示中レイヤー名の集合」をカンマ区切りで保持するため、レイヤー名にカンマがあると壊れます。セットアップ時に検出して結果ダイアログで警告します（リネーム推奨）。

## セットアップ時の自動処理

- **コンポ名の一意化**: グループコンポを `<ルートコンポ名>_<グループ名>` に改名します
  （複数キャラの PSD で「目」「口」コンポが衝突するのを防ぐため）
- **重複レイヤー名の一意化**: 同じコンポ内の同名レイヤーに ` (2)` ` (3)` を付けます
- **デフォルト表情マーカー**: PSD で表示状態だったレイヤーを時刻 0 のマーカーとして書き込みます
  （制御レイヤーにマーカーがないときのみ。これがないと立ち絵が表示されません）

## 目パチ（自動まばたき）

PSD タブの「目パチ」パネルで、マーカー不要の自動まばたきを設定できます

1. 目のコンポをアクティブにして、開き目レイヤーを選択 → `開き目` の「割当」
2. 同様に `閉じ目`（と任意で `中間`）を割当
3. 間隔・速度・ランダム% を調整して `目パチ設定`

**動作の仕組み:**

- time ベースのエクスプレッションで、開き → (中間) → 閉じ → (中間) → 開き を周期的に切替
- 乱数をサイクル番号で固定しているため、開き/中間/閉じの全レイヤーが同じタイミングで同期します
- **表情切替に登録済みのレイヤーに設定すると、現在の表情が「開き目」のときだけまばたきします**
  （笑い目・ジト目などの表情中に不自然にまばたきしません）
- 未登録のレイヤーに設定した場合は常にまばたきします
- 「解除」で表情切替のみの状態（未登録だった場合は不透明度 100）に戻ります

> [!NOTE]
> 口パク（口形状マッピング）設定済みのレイヤーには適用されません（不透明度の競合を防ぐため）

## 再実行（更新）について

`解析してセットアップ / 更新` は何度実行しても安全です（冪等）

- 適用済みのリネーム・制御レイヤー・打ったマーカーはそのまま
- 登録済みレイヤーのエクスプレッションは最新の内容に更新
- 口形状マッピング設定済みのレイヤーは上書きせず保持
- AE 内で手動追加したレイヤーは差分として登録

## 設定（PSD タブ下部）

立ち絵タブの表示・挙動を切り替えられます。設定は `app.settings` で AE 環境に保存され、
プロジェクトを跨いで保持されます。

- **再生ヘッド追従**: 立ち絵タブをアクティブにしたとき、現在時刻の状態に表示を同期
- **「!」常時表示レイヤーを表示**: 立ち絵タブに `!` レイヤーを出すか（グレーアウト表示）
- **非アクティブ階層を隠す**: 上位未選択の階層を、グレーアウトの代わりに非表示にする
- **インデント幅(px)**: 階層のインデント量

## 既知の制限

- **PSD 側の構造変更（レイヤー追加など）は AE の既存コンポに反映されません**
  AE のフッテージ再読み込みはピクセルのみ更新します。構造が変わったら再インポート＋再セットアップが必要で、
  打ったマーカーは旧制御レイヤーに残ります
- **セットアップ後にコンポを改名しないでください**
  エクスプレッションの `comp("名前")` 参照と `[Emo] 名前` の対応が切れます
- 表情切替はグループ（目・眉・口など）ごとに対象コンポを選んでマーカーを打つ方式です
  複数グループをまとめて切り替える場合は、レイヤー選択タブまたは立ち絵タブの「表情セット」を使ってください

> [!IMPORTANT]
> v1.7.0 でレイヤーの **プレフィックス（`*` / `!`）を剥がさず保持** するようになりました。
> v1.6 以前でセットアップ済み（`*` が剥がれている）のコンポは、必要なら PSD から再読み込み →
> 再セットアップしてください。トグル（任意指定）を使うには再セットアップが必要です。


# 立ち絵タブ

PSDToolKit 立ち絵の **階層（目・口・服… さらに 口パク→母音/特殊/簡易→形状）** をまとめて表示し、
各階層を独立して切り替えられるタブです

## 使い方

1. PSD タブで立ち絵をセットアップしておく
2. 立ち絵タブで `立ち絵ルート` を選ぶ（**設定済みのコンポだけ**が一覧に出ます）
3. 目 / 口 / 服 などの階層がインデントで並びます（幅に応じて折り返し）
   - **ラジオ（`*` レイヤー / コンポ）= ラジオボタン**（1 つだけ表示。例: 表情、口の形）
   - **任意指定（無印レイヤー）= チェックボックス**（独立 ON/OFF。例: 頬染め、汗）
   - **常時表示（`!` レイヤー）= チェック済み・グレーアウト**（情報として見えるが操作不可）
4. `∇` / `▸` でサブ階層を折りたためます（折りたたむと配下すべてが隠れます）。縦に溢れたら右端のスクロールバーで上下
5. **上位コンポが選択されていない階層はグレーアウト**します（`*` はコンポにも効きます）

> [!NOTE]
> 再生ヘッドを動かした後は、パネルをクリックして戻ると現在状態に追従します（表示だけ即時更新）。
> 階層の増減など構造が変わったときは「更新」ボタンを押してください
> （ScriptUI は再生ヘッド移動を直接検知できないため）

> [!NOTE]
> 口パクは「口パク」タブで設定します（立ち絵タブには口パク機能はありません）。

## レイヤー種別（PSDToolKit 命名規則）

| 記法 | 意味 | 立ち絵タブでの表示 |
| --- | --- | --- |
| `*名前` | 兄弟内で排他（ラジオ）。コンポにも適用 | ラジオボタン（1 つだけ選択） |
| `名前`（無印） | 任意指定（独立 ON/OFF） | チェックボックス |
| `!名前` | 常に表示 | チェック済み・グレーアウト（操作不可） |

## 仕組み（表示中集合マーカー）

切り替えは制御コンポのマーカーに「**現在表示中のレイヤー名の集合**（カンマ区切り）」として
書き込まれます。これによりラジオ（1 つ）と任意指定（複数同時）を同じ仕組みで扱えます。
「表情セット」は全階層の表示状態をまとめて保存・適用できます（レイヤー選択タブと共有）。

## トラブルシューティング

### レイヤー選択タブ: 登録できない

- 対象コンポがアクティブになっているか確認してください
- `対象` と `制御` の両方が選択されている必要があります

### レイヤー選択タブ: マーカーボタンが押せない

- 制御コンポ内に `[Emo] <対象コンポ名>` の制御レイヤーが必要です
- 先に `制御レイヤー作成` を実行してください

### 口パクタブ: 口パクが反映されない

画像レイヤー名が音素名と一致しているか確認してください  
`[Lab]` レイヤーがあるコンポを `口パク設定` のダイアログで正しく選んでいるか確認してください

### 口パクタブ: lab ファイルが読み込めない

ファイルの文字コードが UTF-8 または Shift-JIS であることを確認してください

### 口パクタブ: 音素レイヤーがないときに何も表示されない

閉じレイヤーなどに「def」を追加してください。一応 pau や sil とは区別しています
